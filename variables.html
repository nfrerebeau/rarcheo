13 Introduction

Une variable statistique désigne une caractéristique mesurable qui peut prendre différentes valeurs. On distingue les variables quantitatives (ou numériques) qui décrivent des caractéristiques quantifiables et les variables qualitatives (ou catégoriques) qui sont des caractéristiques non quantifiables (fig. 13.1). Dans sa forme la plus simple, une variable statistique est représentée par un vecteur atomique.

Figure 13.1: Les types de variables.

Variables quantitatives
Les variables quantitatives prennent des valeurs numériques qui représentent des quantités mesurables. Si les valeurs d’une variable quantitative sont dénombrables et ont un nombre fini de possibilités, cette variable est dite discrète. Les valeurs d’une variable discrète sont souvent, mais pas uniquement, des nombres entiers (la population d’un pays par exemple). À l’inverse une variable quantitative est dite continue si les valeurs qu’elle peut prendre ne sont pas dénombrables et ont un nombre infini de possibilités dans un intervalle donné (la taille et le poids d’un groupe d’individus par exemple).
Variables qualitatives
Les variables qualitatives représentent des caractéristiques non quantifiables, dont les valeurs sont appelées catégories ou modalités90. Une variable qualitative nominale décrit une catégorie sans ordre naturel (le genre, les couleurs sont ainsi des variables nominales). À l’inverse, une variable qualitative est dite ordinale s’il existe une relation d’ordre entre les catégories auxquelles peuvent appartenir ses différentes valeurs (une grandeur mesurée sur une échelle petit/moyen/grand est une variable ordinale).

logical = variable qualitative nominale qui ne peut prendre que deux valeurs : TRUE (vrai) ou FALSE (faux)
